### vless.sh - скрипт по автоматической установке и настройке VLESS сервера на VPS (debian, ubuntu)

```
После заказа VPS рекомендуется сгенерировать пару ключей,  добавить публичный ключ на сервер и отключить удаленный вход по паролю
```


```
1. Подключитесь к серверу по SSH:

bash

ssh root@IP_вашего_сервера

2. Обновите систему:

bash

apt update && apt upgrade -y
apt install curl wget unzip tar socat ufw git -y
```

```
2. Создайте новый файл и скопируйте в него содержимое файла vless.sh:

bash

vi vless.sh

:wq - сохранить изменения

либо сделайте git clone этой страницы и
сделайте файл исполняемым:

bash

chmod +x vless.sh

3. Запустите выполнение файла и следуйте инструкциям на экране:

bash

./vless.sh

4. После завершения исполнения скрипта скопируйте ссылку и отправьте себе в клиент
```

```

Удобный клиент для android V2Box:
Скопировать ссылку на телефоне в буфер
Перейти на вкладку Конфигурация
Нажать "+" в правом верхнем углу приложения
Импортировать v2ray из буфера...
Убедиться, что в центре окна появилась новая запись
Нажать "Подключиться"
Перейти на вкладку Главная и убедиться, что подключение установлено и отображается заданный ip.

```
```
Помните, российское ПО не безопасно и может шпионить за вами.
```
